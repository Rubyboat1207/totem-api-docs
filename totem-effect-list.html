Totem Effect List

AOE Effect:

new AOEEffect(int radius, StatusEffectInstance aoeEffect) new AOEEffect(int radius, new AOEEffect(int radius, ArrayList<StatusEffectInstance> aoeEffect) aoeEffect) AOE Effect (Area of Effect) is a special effect that will apply a status effect to all entities within a certain radius.

Explode Effect:

new ExplodeEffect(int power) Explode Effect is a special effect that will cause the totem to explode with the power specified, whilst not damaging the user.

Freeeze Effect:

new FreezeEffect(int freezeTicks) Cause the user to have the poweder snow frozen effect for the specified amount of ticks.

GiveItem Effect:

new giveItem(ItemStack item) Gives the Player the specified item.

Launch Effect:

new LaunchEffect(double velocity) Launches the user vertically by the specified velocity.

Random Teleport Effect:

new RandomTeleport(float Radius) Teleports the user to a random location within the specified radius.
