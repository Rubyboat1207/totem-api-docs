Your First Totem

Congragulations on setting up totem api correctly! Lets get into the fun part...
To create your first totem you just need to instantiate the class. The constructor is as follows: new TotemItem(Settings settings, Identifier identifier, ArrayList<StatusEffectInstance>, int, @Nullable DeezNuts effect, (Experimental) boolean canUseInVoid)
1. Settings for the item.
2. Identifier for the item, as this will automatically register the item.
3. List of status effects that the totem will give on death.
4. Amount of health remaining the player will have once used.
5. Any non status effect effects that will happen once used.
6. Whether or not the item can be used in the void, leaving this out will result in the value being set to false.

An example of how to use this is as follows: ArrayList effects = new ArrayList<>() { { add(new StatusEffectInstance(StatusEffects.ABSORPTION, 600, 1)); add(new StatusEffectInstance(StatusEffects.REGENERATION, 1200, 0)); add(new StatusEffectInstance(StatusEffects.RESISTANCE, 2400, 0)); } }; @Override public void onInitialize() { new TotemItem(new FabricItemSettings().group(ItemGroup.COMBAT), new Identifier("rubyboat", "cool_totem"), effects, 10, null); } There are many effects, to see the full list of effects, you can go to Totem Effect List to use a special effect just instanciate any class that extends DeezNuts and use the correct constructor. An Example of a totem with a special effect is:
new TotemItem(new FabricItemSettings().group(ItemGroup.COMBAT), new Identifier("rubyboat", "cool_totem"), effects, 10, new ExplodeEffect(3));
