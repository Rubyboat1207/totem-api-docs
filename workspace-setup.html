Workspace Setup

Get your fabric environment setup correctly, if you dont know how to do this, please refer to the Fabric Getting Started Guide
Assuming you are already setup, you can now proceed to getting totem API setup.
To begin put this in your build.gradle file:
repositories { maven { url 'https://jitpack.io' } } dependencies { modImplementation "com.github.Rubyboat1207:totem-api:${project.totem_api_version}" } put this in your gradle.properties file: totem_api_version= This is the latest, but if you would like a different version of totem api, you can get the version here
Then finally put this inside your fabric.mod.json file: "depends": { "totem_api": "*" }
