Your First Effect

To create an effect, simply create a new class that extends the DeezNuts class. public class MyEffect extends DeezNuts { } There are 3 overridable methods that you can use to customize your effect. void OnDeath(LivingEntity user, World world, ItemStack stack) ActionResult onClick(LivingEntity user, World world, BlockPos selectedBlock, ItemStack stack) String getTooltip() OnDeath is called when the user dies.
onClick is called when the user clicks on a block.
getTooltip is called whenever the player hovers over the item in their inventory. and it looks like this in game:
public String getTooltip() {return "launches you with a force of " + velocity;}
